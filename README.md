# OIDC Game
Learn how to use Open ID Connect via this simple interactive game

## Getting started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
